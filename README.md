# Qaders Heaven - Property Management System

A modern property management website built with React, Vite, and TailwindCSS.

## Features

- Home Dashboard
- Admin Management
- Report Generation
- Fund Management
- Service Provider Contact
- Property Management
- Floor Management
- Meter Management
- Unit Entry
- Property Status
- Tenants Details
- Tenant's Highlights
- Tenant's Bill Generate
- Tenants Bill
- Payment Details
- Settings & Basic Settings

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router DOM
- Lucide React (Icons)
